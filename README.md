# BusMaps - Telegram Mini App

Telegram Mini App для построения маршрутов с использованием Leaflet карт.

## Описание

Приложение для построения пеших маршрутов и маршрутов общественного транспорта (автобусы и т.п.) в Telegram.

## Технологии

- React 18 + TypeScript
- Vite
- Leaflet + React-Leaflet
- Telegram WebApp SDK
- OpenStreetMap API
- OSRM для построения маршрутов

## Установка

```bash
npm install
```

## Разработка

```bash
npm run dev
```

## Сборка

```bash
npm run build
```
